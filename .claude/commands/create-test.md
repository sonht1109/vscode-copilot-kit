## Argument

- `[function]`: The function or code snippet for which you want to create tests.
- `--u`: Optional flag to indicate that you want to create unit tests specifically. Default will be --u flag.
- `--i`: Optional flag to indicate that you want to create integration tests specifically.

## Core Principles

- Analyze and use `test-writing` skill to create effective and comprehensive tests for the provided code.
- After done, review test cases again using `/review-test` command.
